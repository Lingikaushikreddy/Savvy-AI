# Voice Coaching

This module provides real-time analysis of sales calls.

## Architecture
- `AudioCaptureManager`: Captures raw audio.
- `VoiceCoach`: Analyzes audio chunks for sentiment and speech patterns.
- `CoachingOverlay`: Displays feedback to the user.

## metrics
- Pace (WPM)
- Sentiment (Positive/Negative/Neutral)
- Objection Handling
